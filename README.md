# URL Album 2.1

**Portable менеджер закладок для Windows 7 SP1 / 8.1 / 10 / 11**

Духовный наследник классического URL Album (~2008). Работает на всех версиях Windows начиная с 7 SP1 — без WebView2, без браузера, без зависимостей.

---

## Скачать

**[Релиз v2.0.1 на GitHub](https://github.com/skljar/url-album-2/releases/latest)** — ≈6.5 МБ, portable

Распакуй и запусти `URL-Album-2.0.1.exe`. Ничего не устанавливается.

---

## Возможности

### Навигация
- Дерево папок с раскрытием/закрытием
- Список ссылок с favicon иконками
- Resizable панели — тащи разделитель
- Поиск в реальном времени по названию, URL, заметке (Ctrl+F)
- Навигация стрелками ↑↓ по списку ссылок

### Работа со ссылками
- Создание, редактирование (F4), переименование (F2), удаление (Del)
- Заметки к ссылкам
- Двойной клик → открыть в браузере
- Копирование URL в буфер обмена (правый клик)
- Сортировка: по названию или URL (asc/desc)
- Проверка доступности ссылок (HTTP HEAD)

### Favicon
- Загрузка favicon в фоне (3 стратегии: /favicon.ico → HTML → DuckDuckGo)
- Кэш рядом с exe в `Data/favicons/`

### Контекстное меню (правый клик)
- Папка: открыть, новая подпапка, новая ссылка, переименовать, удалить
- Ссылка: открыть URL, свойства, копировать URL, переименовать, удалить

### Импорт
- Из старого URL Album (`ua.dat`, кодировка Windows-1251)
- Из браузера: Chrome, Edge, Brave, Opera (файл `Bookmarks` JSON)
- Из HTML (Netscape bookmark format — экспорт из любого браузера)

### Экспорт
- В HTML (Netscape format, открывается в браузерах)
- В TXT (URL + название)

### Работа с базой
- Несколько баз данных — открыть/создать через диалог
- Автовозобновление последней открытой БД при запуске
- Резервная копия БД

## Горячие клавиши

| Клавиша | Действие |
|---|---|
| F4 | Свойства ссылки |
| F2 | Переименовать |
| Del | Удалить |
| Enter | Открыть URL |
| Ctrl+F | Фокус на поиск |
| ↑ / ↓ | Выбор в списке |
| Двойной клик | Открыть URL |

## Установка

1. Скачайте ZIP со [страницы релизов](https://github.com/skljar/url-album-2/releases/latest)
2. Распакуйте в любую папку
3. Запустите `URL-Album-2.0.1.exe`

### Файлы рядом с exe

```
URL-Album-2.0.1.exe
album.db          ← база закладок
settings.json     ← ширина панели
last_db.txt       ← последняя открытая база
Data\favicons\    ← кэш favicon
```

## Отличие от URL Album 2

| | URL Album 2 | URL Album 3 |
|---|---|---|
| Минимальная ОС | Windows 10 | Windows 7 SP1 |
| Движок UI | WebView2 (Edge) | Slint (свой) |
| Скриншоты | Edge/Chrome headless | — (в разработке) |
| Размер | 3.7 МБ | 6.5 МБ |

## Технологии

- [Slint](https://slint.dev) — UI фреймворк (без браузера)
- Rust — backend
- SQLite (rusqlite, bundled)
- reqwest — HTTP для favicon/проверки ссылок
