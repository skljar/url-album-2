# URL Album 2 — Полный список функций

## Навигация
- [x] Дерево папок в левой панели (раскрытие/закрытие)
- [x] Список ссылок в правой панели (compact list view)
- [x] Клик по папке — выделение и показ содержимого
- [x] Двойной клик по папке — раскрыть/свернуть
- [x] Клик по ссылке — карточка (detail view) с превью и заметкой
- [x] Двойной клик по ссылке — открыть в браузере
- [x] ↑↓ стрелки — перемещение по дереву
- [x] Папки всегда выше ссылок в дереве и в списке
- [x] При старте: все папки закрыты, первая папка подсвечена
- [x] Splitter между деревом и списком (resizable, сохраняется)
- [x] Resizable колонка "Название" в списке (сохраняется)
- [x] Breadcrumb — путь к текущему элементу

## База данных
- [x] Создать новую базу (Файл → Создать базу)
- [x] Открыть существующую базу (Файл → Открыть базу)
- [x] Последние базы — подменю с историей (до 10 баз)
- [x] Закрыть базу — вернуться на стартовый экран
- [x] Свойства базы — путь, размер, кол-во папок/ссылок
- [x] Очистить базу — удалить всё содержимое
- [x] Резервная копия без скриншотов
- [x] Резервная копия со скриншотами
- [x] Несколько баз — переключение без перезапуска
- [x] Авто-resume — при старте открывает последнюю базу
- [x] WAL checkpoint при закрытии и переключении

## Папки
- [x] Создать папку (Ссылки → Новая ссылка или контекстное меню)
- [x] Создать подпапку внутри папки (контекстное меню → Новая папка)
- [x] Переименовать папку — inline прямо в дереве (F2)
- [x] Удалить папку рекурсивно со всем содержимым
- [x] Свойства папки — название, кол-во ссылок, подпапок, путь
- [x] Drag & Drop — перетащить папку в другую папку
- [x] Сортировка папки: по имени, URL, дате (asc/desc, toggle)
- [x] Глобальная сортировка всех папок (toolbar)

## Ссылки
- [x] Создать ссылку — диалог с URL, названием, заметкой
- [x] Редактировать ссылку (F2 / Ссылки → Редактировать)
- [x] Удалить ссылку (Del)
- [x] Открыть в браузере по умолчанию (Enter / двойной клик)
- [x] Открыть в переносном браузере (первый из списка)
- [x] Открыть с помощью... — выбор браузера
- [x] Копировать URL (Ctrl+C)
- [x] Свойства ссылки (F4)
- [x] Drag & Drop — переместить ссылку в папку
- [x] Авто-добавление https:// при открытии (без изменения в БД)

## Скриншоты сайтов (рисунки)
- [x] Обновить рисунок — скриншот одной ссылки (правый клик)
- [x] Обновить рисунки — пакетное обновление для папки (правый клик)
- [x] Прогресс-панель с счётчиком и кнопкой Отмена (перетаскивается)
- [x] Открыть рисунок — открыть файл скриншота (F12)
- [x] Удалить рисунок — очистить скриншот ссылки
- [x] Настройки скриншота: размер (ширина×высота), таймаут (сек)
- [x] Принудительное завершение браузера по таймауту (kill процесса)
- [x] 3 параллельных потока для пакетного обновления
- [x] Требуется Edge или Chrome

## Иконки сайтов (favicon)
- [x] Загрузить favicon одной ссылки (правый клик)
- [x] Загрузить favicon'ы — пакетная загрузка для папки (рекурсивно)
- [x] Прогресс-панель favicon с кнопкой Отмена (перетаскивается)
- [x] 4 стратегии поиска: favicon.ico → HTML link → DuckDuckGo → Google
- [x] Domain dedup — один favicon на домен, применяется ко всем ссылкам
- [x] Кэш в Data\favicons\ — не скачивается повторно
- [x] Favicon в дереве, в списке, в карточке
- [x] После загрузки — автоматическое обновление дерева и списка

## Импорт (меню Перенос)
- [x] Импорт из браузера — Chrome, Firefox, Edge, Opera, Brave (автодетект)
- [x] Импорт из HTML файла (Netscape bookmarks format)
- [x] Импорт из TXT файла (одна ссылка на строку)
- [x] Импорт из файла синхронизации (JSON)
- [x] Импорт из ua.dat (оригинальный URL Album)
- [x] Импорт в конкретную папку (правый клик по папке → Импорт в папку)

## Экспорт (меню Перенос / контекстное меню папки)
- [x] Экспорт папки в HTML файл
- [x] Экспорт папки в TXT файл
- [x] Экспорт в файл синхронизации с рисунками
- [x] Экспорт в файл синхронизации без рисунков

## Поиск
- [x] Поиск по названию, URL, заметке (Ctrl+F)
- [x] Результаты в отдельной панели
- [x] Поиск дубликатов ссылок — full-screen двухпанельный finder
- [x] Удаление дублей по одному или оставить одну

## Проверка ссылок
- [x] Проверить все ссылки в папке (правый клик → Проверить ссылки)
- [x] HTTP-статус каждой ссылки (200, 404, таймаут и т.д.)
- [x] Пауза / Остановка проверки
- [x] Ссылки на папки в дереве — Ссылки → Проверить ссылки

## Браузеры
- [x] Автодетект установленных браузеров
- [x] Ручное добавление браузеров (Перенос → Браузеры)
- [x] Portable browsers.json рядом с exe
- [x] Открыть ссылку в любом из настроенных браузеров

## Интерфейс
- [x] Toolbar с настраиваемым набором кнопок (drag & drop)
- [x] Меню: Файл / Ссылки / Перенос / Поиск / Вид
- [x] Контекстное меню ссылки (правый клик)
- [x] Контекстное меню папки (правый клик в дереве и списке)
- [x] Развернуть / Свернуть все папки (меню Вид или toolbar)
- [x] Настройки: тема, toolbar, ширина колонки, splitter, скриншоты
- [x] Меню закрывается при клике на titlebar или Alt+Tab

## Portable
- [x] Все данные рядом с exe (album.db, settings.json, browsers.json и т.д.)
- [x] Ничего не пишется в реестр (только читается для поиска браузеров)
- [x] Можно запускать с флешки
- [x] Размер exe: ~8 MB
